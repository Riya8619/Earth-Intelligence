# HackHazards Frontend

This is the frontend application for the HackHazards dashboard.

## Local development

1. Change to the `frontend` folder
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Open the app in your browser at `http://localhost:5173`

## Build

Run `npm run build` to produce a production-ready bundle.

## Preview

Run `npm run preview` to preview the built app locally.

## Notes

This project uses Vite, React, Tailwind CSS, and React Query. Configure the backend API URL in your environment or proxy settings as needed.
